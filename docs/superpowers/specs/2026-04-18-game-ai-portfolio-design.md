# Game AI Portfolio Design

## Goal

Update the personal website to present bobozai2019 as a game AI programmer, with `godot-project-analysis` as the lead project.

## Direction

Use the approved "dark technical lab" direction:

- Lead with Godot AI tooling, architecture recovery, and AI-assisted game development.
- Use a dark, technical visual system with grid-like structure, terminal-style labels, and sharper project hierarchy.
- Keep the site static and lightweight: only `index.html`, `styles.css`, a small verification script, and documentation.

## Content

The homepage should:

- Put the game AI programmer positioning in the hero.
- Feature `Godot Project Analysis` as the first and strongest work item.
- Keep `godot_2d_test_game` as supporting proof of Godot game development.
- Replace vague placeholder project text with more concrete AI workflow and prototype language.
- Include GitHub links for the profile and both visible projects.

## Visual Design

Use a dark charcoal base with green, coral, and cyan accents. Avoid a flat single-color theme. Add stronger spacing, labels, capability chips, and a signal/graph-style hero panel to make the page feel less plain while staying professional.
